# To-Do List Web App (Learning Project)

This is a simple To-Do List web application built as a learning project. It demonstrates basic HTML, CSS, and JavaScript skills for beginners.

## Features
- Add tasks to your to-do list
- Mark tasks as completed by clicking on them
- Remove individual tasks by right-clicking
- Remove all tasks with a single button
- Responsive and colorful UI

## How to Use
1. Open `index.html` in your web browser.
2. Type a task in the input box and click **Add Task** (or press Enter).
3. Click a task to mark it as completed (strikethrough effect).
4. Right-click a task to remove it from the list.
5. Click **Remove All Tasks** to clear the entire list.

## Project Structure
- `index.html` — Main HTML file with the app layout
- `style.css` — Styles for the app, including completed task effects
- `script.js` — JavaScript logic for adding, completing, and removing tasks

## Purpose
This project is for educational purposes only. It is designed to help beginners learn the basics of web development, including DOM manipulation, event handling, and styling.

Feel free to modify and experiment with the code to enhance your learning!
